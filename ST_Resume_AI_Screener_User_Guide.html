评分体系Scoring System

5 维度雷达图评分5-Dimension Radar Scoring

五个维度以雷达图直观展示，权重可在侧边栏自定义（总和须 100%）。AI 配置 JD 后自动建议最优分布。对比模式下最多 3 人雷达叠加，AI 自动生成对比分析。Five dimensions displayed on a radar chart. Weights are customizable in the sidebar (must sum to 100%). Comparing mode overlays up to 3 candidates with AI-generated comparative analysis.

常见问题FAQ

常见问题解答Frequently Asked Questions

支持哪些文件格式？What file formats are supported?
+
PDF、DOCX、DOC 和 TXT。PDF 推荐以获得最佳解析效果，DOCX 不提供内联预览（可下载）。PDF, DOCX, DOC, and TXT. PDF recommended for best parsing. DOCX not previewed inline (download only).
雷达图和对比功能如何使用？How to use radar & compare?
+
展开候选人卡片即可看到 5 维雷达图。点击结果页右上角 Compare 进入对比模式，勾选 2-3 人自动生成叠加雷达图和 AI 对比分析。Expand any card to see the 5-dim radar. Click Compare in the top-right, select 2-3 candidates for overlay radar and AI analysis.
如何分享给面试官？How to share with interviewers?
+
在结果页点击 Share 按钮，链接自动复制。面试官打开后看到 JD + 评分排名（只读）。可在服务端 API 随时撤销任意令牌。Click Share on the results page — link auto-copies. Interviewers see JD + scores (read-only). Tokens can be revoked anytime via API.
AI 对比分析使用什么模型？Which model for comparison analysis?
+
优先调用 Gemini 2.5 Flash，如失败则自动降级到 DeepSeek-chat。对比分析只生成一次不会反复刷新。Gemini 2.5 Flash first, with automatic fallback to DeepSeek-chat. Analysis generates once — no repeated refreshes.
如何切换中英文？How to switch languages?
+
页面右上角 中文/EN 按钮。切换后 UI 和 AI 分析结果同步更改。Use the 中文/EN toggle in the top-right. UI and AI results switch instantly.
