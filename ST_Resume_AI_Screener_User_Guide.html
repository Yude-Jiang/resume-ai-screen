操作指南Walkthrough

设置 JD & 上传简历Setup JD & Upload

01

设置职位描述 (JD)Set Up Job Description

  • 左侧面板粘贴 JD 文本，或上传 JD 文件（PDF / DOCX / TXT）Paste JD text or upload a JD file
  • 点击 Auto-configure from JD，AI 自动提取职位名称、建议权重和硬性门槛Click Auto-configure — AI extracts job title, weights, thresholds
  • 侧边栏展开 Scoring Weights 微调滑块（默认展开，总和须 100%）Fine-tune weight sliders in sidebar (must sum to 100%)
💡 提示：Tip: JD 越详细权重建议越精准，建议包含 3–8 个必备技能标签。More detailed JDs yield more accurate suggestions. Include 3–8 required skills.
02

上传简历 & 开始筛选Upload Resumes & Screen

  • 右侧虚线区域拖拽或点击上传（PDF / DOCX / TXT，最多 50 份）Drag or click to upload resumes in the right panel (max 50)
  • 点击底部 Screen Candidates → 按钮启动分析Click the Screen Candidates → button to start analysis
  • 4 并发处理，每份约 3-10 秒，50 份约 1-2 分钟4 concurrent, ~3-10s per resume, ~1-2 min for 50

查看评分详情View Score Details

多人雷达叠加对比Multi-Candidate Compare NEW

04

选择候选人进行叠加对比Select candidates to compare

  • 点击顶部 Compare 进入对比模式，勾选 2-3 位候选人Click Compare button, select 2-3 candidates
  • 自动生成雷达叠加图（左）+ AI 对比分析（右），三色多边形交叉重叠清晰可见Auto-generated overlay radar (L) + AI comparison (R) with 3-color polygon overlap
  • AI 对比分析按【综合对比】【各候选人分析】【推荐排序】三段式结构化呈现AI analysis structured as [Summary] [Per Candidate] [Ranking]
💡 提示：Tip: 对比分析只生成一次，更改选中候选人时才会重新生成。不会因数据轮询刷新。Analysis generates once — only re-triggers when selection changes, not on data polling.

分享给面试官Share to Interviewer NEW

05

一键生成分享链接Generate share link

  • 点击结果页右上角 Share，自动生成唯一令牌并复制链接到剪贴板Click Share — generates unique token and copies link to clipboard
  • 面试官打开链接看到：岗位 JD + 所有候选人评分排名 + 详情展开（只读，无操作权限）Interviewer sees: JD + ranked candidates + expand details (read-only)
  • 每个岗位可生成多个令牌，可随时单独撤销Multiple tokens per job — revoke individually anytime

辅助功能一览Additional features

  • 仪表盘：4 列指标卡片（累计简历·节省工时·平均分·活跃岗位）+ 漏斗行 + 岗位卡片Dashboard: 4-column metrics + funnel + job cards

  • 盲筛模式：开关后姓名/学历/经验用 CSS filter 模糊化（GPU 加速，无闪屏），雷达图与 AI 分析保持清晰Blind mode: CSS blur on personal info — no DOM flash, radar & AI remain clear

常见问题FAQ

常见问题解答Frequently Asked Questions

支持哪些文件格式？What file formats?
+
PDF、DOCX、DOC 和 TXT。PDF 推荐获得最佳解析效果，DOCX 不提供内联预览（可下载）。PDF, DOCX, DOC, TXT. PDF recommended. DOCX not previewed inline (download available).
雷达图和对比功能如何使用？Radar & compare?
+
展开候选人卡片即看到 5 维雷达图。点击 Compare 进入对比模式，勾选 2-3 人自动生成叠加雷达图与 AI 对比分析。Expand card for 5-dim radar. Click Compare, select 2-3 for overlay + AI analysis.
如何分享给面试官？Share to interviewer?
+
结果页点击 Share，链接自动复制。面试官打开后看到 JD + 评分（只读）。可随时撤销令牌。Click Share on results page — link auto-copies. Interviewer sees JD + scores (read-only). Tokens revocable.
权重必须设为 100% 吗？Weights must be 100%?
+
是的。侧边栏实时显示当前总和，不满足时红色警告，筛选无法启动。Yes. Sidebar shows live total — red warning if not 100%.
盲筛模式会闪屏吗？Blind mode flash?
+
不会。采用 CSS filter: blur() 而非文本替换，GPU 加速 500ms 平滑过渡，雷达图与 AI 分析始终清晰。No. Uses CSS blur (GPU-accelerated) with 500ms transition — no DOM flash. Radar and AI analysis remain clear.
