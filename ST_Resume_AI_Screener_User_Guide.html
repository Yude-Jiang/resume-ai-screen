产品概述Overview

为 ST HR 团队打造的智能筛选工具Built for ST HR Teams

通过 Gemini / DeepSeek 大模型对候选人简历进行多维度雷达图自动评分，支持最多 3 人叠加对比、一键生成令牌分享给面试官，所有 UI 与 AI 输出中英双语同步切换。评分维度可在侧边栏自由增减、自定义权重。LLM-powered multi-dimension radar scoring with up to 3-candidate overlay comparison and one-click token sharing. Dimensions and weights are fully customizable in the sidebar. All UI and AI output switches between Chinese and English.

操作指南How to Use

完整操作流程Complete Workflow

01

设置 JD & 上传简历Setup JD & Upload

  • 左侧面板粘贴 JD 或上传文件（PDF/DOCX/TXT），AI 自动提取权重和阈值Paste JD or upload file — AI auto-extracts weights and thresholds
  • 右侧拖拽上传简历（最多 50 份），点击底部 Screen Candidates → 启动Drag resumes in right panel (max 50), click Screen Candidates to start
  • 4 并发处理，每份约 3–10 秒，50 份约 1–2 分钟4 concurrent, ~3-10s each, ~1-2 min for 50
02

查看雷达图 & AI 推荐Radar & AI Recommendation

  • 筛选完成自动跳转结果页，候选人按分数降序Results page auto-opens, sorted by score descending
  • 点击卡片展开 — 左侧雷达图 + 右侧 AI 推荐意见 各占 50% 宽度Expand card — radar chart (L) + AI recommendation (R) at 5:5 ratio
  • 下方展示亮点与不足；底部 PDF 预览（PDF 内联，DOCX 提供下载）Below: highlights & gaps; bottom: PDF preview (inline) or DOCX download
03

多人雷达叠加对比Multi-Candidate Compare

NEW
  • 点击顶部 Compare 进入对比模式，勾选 2-3 人Click Compare, select 2-3 candidates
  • 自动生成雷达叠加图（左）+ AI 对比分析（右），三色多边形交叉重叠Auto overlay radar (L) + AI comparison (R) with 3-color polygon overlap
  • AI 分析按【综合对比】【各候选人分析】【推荐排序】三段式呈现AI structured as [Summary][Per Candidate][Ranking]
💡 提示：Tip: 对比分析只生成一次，更改选中候选人才会重新生成。不会因轮询而刷新。Analysis triggers only on selection change — not on data polling.
04

分享给面试官Share to Interviewer

NEW
  • 点击结果页右上角 Share，自动生成唯一令牌并复制链接到剪贴板Click Share — generates token and copies link to clipboard
  • 面试官看到：岗位 JD + 所有候选人评分排名 + 详情展开（只读，无操作权限）Interviewer sees: JD + ranked candidates + expand details (read-only)
  • 同一岗位可为不同面试官生成多个令牌，作废任意一个不影响其他Create multiple tokens for different interviewers per job — revoke any one without affecting others
05

HR 复核 · 仪表盘 · 人才库Review · Dashboard · Library

  • 人工调分：展开卡片底部 Adjust Score，输入分数和原因，保存后标记 VERIFIEDOverride: Adjust score at card bottom, saved as VERIFIED
  • 仪表盘：4 列指标卡 + 筛选漏斗 + 岗位卡片，全局数据一目了然Dashboard: 4-column metrics + funnel + job cards
  • 人才库：跨职位搜索、分层过滤（Strong / Pending / Rejected），点击跳转详情Talent Library: Cross-job search & filter
  • 删除岗位：侧边栏 🗑️ 或仪表盘卡片可删除岗位及关联结果Delete: 🗑️ in sidebar or dashboard cards
  • 盲筛模式：一键 CSS filter 模糊敏感信息（GPU 加速，无闪屏），雷达图与 AI 分析保持清晰Blind mode: CSS blur sensitive fields (GPU-accelerated, no flash)
  • 语言切换：右上角 中文/EN 按钮，UI 与 AI 分析结果同步切换Language: 中文/EN toggle in top-right, UI & AI sync

支持哪些文件格式？What file formats?
+
PDF、DOCX、DOC 和 TXT。PDF 推荐以获得最佳解析效果，DOCX 不提供内联预览（可下载）。PDF, DOCX, DOC, and TXT. PDF recommended. DOCX not previewed inline (download available).
雷达图和对比功能如何使用？Radar & compare?
+
展开候选人卡片即看到多维度雷达图。点击 Compare 进入对比模式，勾选 2-3 人自动生成叠加雷达图与 AI 对比分析。Expand card for radar chart. Click Compare, select 2-3 for overlay + AI analysis.
如何分享给面试官？Share to interviewer?
+
结果页点击 Share，链接自动复制。面试官打开后看到 JD + 评分排名（只读）。同一岗位可给不同人发不同链接，作废某个链接不影响其他人。Click Share on results page — link auto-copies. Interviewer sees JD + scores (read-only). Create unique links per interviewer — revoke any one individually without breaking others.
维度可以自定义吗？Customizable dimensions?
+
可以。侧边栏 + 按钮添加新维度，点击标签重命名，权重总和须 100%。Yes. Use + to add dimensions, click labels to rename. Weights must sum to 100%.
